# NestJS Project

Этот проект представляет собой пример приложения, построенного с использованием фреймворка NestJS.

## Содержание

- [Установка](#установка)
- [Запуск](#запуск)
- [Миграция на Yarn 3](#миграция-на-yarn-3)

## Установка

1. Установите зависимости:

   ```bash
   yarn
   ```

## Запуск

1. Запустите приложение в режиме разработки:

   ```bash
   yarn start:dev
   ```

2. Приложение будет доступно по адресу `http://localhost:3000`.

## Миграция на Yarn 3

1. **Установите Yarn 3:**

   Следуйте инструкциям на [официальном сайте Yarn](https://yarnpkg.com/getting-started/install) для установки Yarn 3.

2. **Инициализируйте Yarn 3:**

   В корне вашего проекта выполните команду:

   ```bash
   yarn set version berry
   ```
3. **Установите зависимости:**

   Удалите старые зависимости и установите их заново с помощью Yarn 3:

   ```bash
   rm -rf node_modules
   yarn install
   ```
