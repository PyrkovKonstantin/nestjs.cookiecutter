# NestJS Entity module Cookiecutter
A NestJS cookiecutter providing a simple backen service.

/home/kopyrkov/projects/nestjs.cookiecutter/{{ cookiecutter.entity_slug }}/src## Requirements / assumptions in this module
This module assumes you are using the following libraries
- TypeORM
- class validator (Registered as a global interceptor)

## How to use
Make sure you have [Cookiecutter installed](https://cookiecutter.readthedocs.io/en/1.7.2/installation.html)
```shell
$ cookiecutter https://github.com/PyrkovKonstantin/nestjs.cookiecutter
```
